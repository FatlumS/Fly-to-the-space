🔐 Verify Fairness

Cryptographically verify any round's crash point

How it works: Every round is generated using your server seed, a client seed, and a nonce. You can independently verify that the crash point was calculated fairly and not manipulated.
Verifying...
Calculation Formula:
combined = serverSeed + '-' + clientSeed + '-' + nonce
hash = SHA256(combined)
first13Hex = hash.substring(0, 13)
randomInt = parseInt(first13Hex, 16)
random = randomInt / (2^52 - 1)
crashPoint = (1 / (1 - random)) * 0.96
